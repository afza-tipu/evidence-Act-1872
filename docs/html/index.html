🔍 About the Project

This open-source project digitizes the entire Evidence Act, 1872 in multiple formats for legal professionals, students, developers, and the general public. You’ll find sections, definitions, and articles in:

  • ✅ PDF and Markdown versions of the Act
  • ✅ Searchable CSV and JSON datasets
  • ✅ SEO-optimized legal articles with MLA citations
  • ✅ Google search snippet insights for each section
